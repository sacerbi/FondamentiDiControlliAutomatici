# Esercizi del libro "Fondamenti di Controlli Automatici"

Questo repository contiene le mie implementazioni in [**Octave**] e [**C++**] degli esercizi presentati nel libro di testo "Fondamenti di Controlli Automatici, 4e".

L'obiettivo è fornire una risorsa di codice per chiunque stia studiando la materia, in modo da poter visualizzare e testare le soluzioni degli esercizi.

---

### Riferimento Bibliografico

Gli esercizi e i problemi originali sono tratti da:

* **Bolzern, P., Scattolini, R., e Patrizio, B. (2015). *Fondamenti di Controlli Automatici*. 4a edizione, McGraw-Hill Education.**

Il codice in questo repository è una mia personale implementazione e non rappresenta le soluzioni ufficiali del libro.

---

### Struttura del Repository

Il codice è organizzato per capitolo, in modo da riflettere la struttura del libro. Ogni cartella corrispondente a un capitolo contiene i file di codice relativi agli esercizi trattati.

Ad esempio:

* `capitolo_1/`
    * `octave/`
        * `esercizio_1_2.m`
    * `cpp/`
        * `esercizio_1_2.cpp`
* `capitolo_2/`
    * `...`
